My work

Sesquipedalian Dictionary

This is my most recent work, is a dictionary app that allows you to search any word in the english language. This App does not only give you the meaning and phonetics (in audio and writting) of a word but also shows images related to the searched word. For this end result were used two different API's, the Google Dictitionary API and Pexels API. To give the app a bit of a fun interaction I built a 3d owl taht moves with the cursor using Spline.

Ducky Duck Weather App

A real-time weather web app built from scratch. The connection to the Openweather API allows for the app to show the current weather at your current location or to search for the forecast of any city in the world. To create a more interesting interface I designed Ducky Duck in various weather conditions and using JavaScript combined with CSS you can see the colors and the character change accordingly to the weather conditions.

Ballet Project

This was my first project for SheCodes, the assignment was to building a simple landing page using HTML 5, CSS and a sprinkle of JavaScript, about something we were passionate about, I chose ballet. In this page you can see the simulation of booking a class, when you click in the button, it will ask you to input some info and will give you the info you need.
